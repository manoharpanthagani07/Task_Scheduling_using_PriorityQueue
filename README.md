# Task_Scheduling_using_PriorityQueue
Task_Scheduling_using_PriorityQueue  A web-based Task Scheduler &amp;amp; HR Management Suite using a priority queue. Managers can assign and track tasks, approve employees, manage salaries, and monitor performance. Employees can view tasks, chat with managers, and track progress. Includes analytics and real-time chat.

# Installation & Setup

# Technology Stack
   HTML/CSS – User interface and styling.
   JavaScript – Dynamic functionality including task scheduling, employee management, and chat.
   Chart.js – Analytics and visual representation of tasks.
   Priority Queue Logic – For task prioritization and scheduling.
   Client-side Storage – Temporary simulation of a database (all data resets on page reload)
# How It Works
 New Employee Registration
 Employees fill in a registration form.
 Requests are sent to the manager for approval.
 Approved employees gain access to the system.
 Manager Dashboard
 Approve or deny new employee requests.
 Assign tasks to employees with deadlines and priorities.
 Monitor pending and completed tasks.
 Manage employee ranks and salary.
 Access real-time analytics and charts.
 Employee Dashboard
 Login with approved credentials.
 View assigned tasks and their priorities.
 Chat with manager for updates or queries.
 Track personal progress and completed tasks.
 Task Execution
 Tasks are organized in a priority queue.
 
# User-Friendly UI
 Responsive and clean interface.
 Background blur effect for professional look.
 Smooth animations and hover effects for tasks and employee cards.
 Highest priority tasks are executed first.
 Task completion updates analytics and employee dashboards automatically.
